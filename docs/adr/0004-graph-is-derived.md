# ADR-0004: Graph is derived

- Status: Accepted
- Date: 2026-07-18

## Context and decision

Inferred relationships can be wrong and must not redefine source meaning. Original revision text and deterministic structure remain canonical; any graph is derived, evidence-backed, versioned, bounded, and removable.

## Alternatives and consequences

A graph-first model can simplify traversal but raises correctness and operational complexity before measured value exists. Deferral preserves reversibility; graph retrieval must later beat lexical/semantic baselines in ablation tests.
