# Vision

Omni-Sem provides fast, private, source-grounded local context to compatible AI agents. Users explicitly approve roots; source files remain authoritative and immutable; derived data is disposable.

The initial product proves deterministic Markdown indexing and lexical retrieval before semantic embeddings or relationships. It is not a chat client, agent, file manager, editor, cloud platform, or general graph database. Trust comes from precise provenance, revision identity, bounded output, and graceful lexical fallback.
